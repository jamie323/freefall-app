# Freefall

Premium one-touch physics puzzle game for iOS.

**Tap to flip gravity. Navigate through obstacles. Reach the goal.**

- No ads
- No lives
- No IAP
- No subscription
- One-time purchase
- Offline-first

- No backend, no dependencies
